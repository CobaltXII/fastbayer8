# fastbayer8

Fast 8-color Bayer dithering written in C++11.

# Compilation

```bash
# Multithreaded version
./build.sh
# Non-multithreaded version
./build_no_omp.sh
```

# Usage

```bash
cobalt$ ./fastbayer8.o
usage: ./fastbayer8.o <file>
```
